**Zod Schemas (per RESEARCH.md Pattern 1):**

Define two Zod schemas at module level:

1. `matchReportSchema` -- z.object with: score (number 0-100), summary (string), strengths (array of strings), gaps (array of strings), keywords (object with matched/missing/bonus each array of strings), sections (object with summary/skills/experience/projects/education each having matchRate 0-1, matchedItems array, missingItems array, summary string).

2. `suggestionSchema` -- z.object with: id (string), section (enum of summary/skills/experience/projects/education), type (enum of add/modify/remove), current (nullable string), suggested (string), reason (string). The generateSuggestions function validates against z.array(suggestionSchema).

**Model helper:**
```
function getModel() {
  const modelName = process.env.ANALYSIS_MODEL || 'gemini-2.5-flash'
  return google(modelName)
}
```

**analyzeResume(resume, jobPosting):**
- Build a prompt that includes the full resume JSON and jobPosting.posting_text
- Call `generateObject({ model: getModel(), schema: matchReportSchema, prompt })`
- Return the validated object directly
- Catch `LoadAPIKeyError` and re-throw with message: "AI analysis requires GOOGLE_GENERATIVE_AI_API_KEY. Set it in your .env file or use the heuristic provider."
- Catch `APICallError` and re-throw with sanitized message (do NOT include API key in message -- per T-11.5-04)
- Catch `NoObjectGeneratedError` and re-throw with: "AI returned invalid analysis format. Falling back to heuristic."

**generateSuggestions(resume, report):**
- Build a prompt with the resume JSON and report JSON, asking for actionable improvement suggestions as structured patches
- Call `generateObject({ model: getModel(), schema: z.array(suggestionSchema), prompt })`
- Return the validated array
- Same error handling as analyzeResume

**require() imports (per RESEARCH.md Pitfall 2 -- CommonJS works):**
```
const { generateObject, LoadAPIKeyError, APICallError, NoObjectGeneratedError } = require('ai')
const { google } = require('@ai-sdk/google')
const { z } = require('zod')
```

**Prompt design for analyzeResume:**
"You are a resume reviewer. Compare this resume against the job posting and return a MatchReport. Be strict -- only count a keyword as matched if the resume genuinely demonstrates competence. Provide detailed section summaries citing specific resume content."

**Prompt design for generateSuggestions:**
"Based on this MatchReport, generate actionable suggestions to improve the resume. Each suggestion must be a structured patch (add, modify, or remove) with clear reasoning referencing actual resume content."

Do NOT use generateText -- use generateObject exclusively (per RESEARCH.md Anti-Patterns).
