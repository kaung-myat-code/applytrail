# Plan 12-02: Frontend — Generate Button Wiring + Preview Page + Routing

## Goal

Enable the "Generate Tailored Resume" button on the review page, persist accept/reject decisions as a server-side draft, and create a dedicated preview page where the user can review the full tailored resume, edit its name, and save it to the library.

## Requirements

TAILOR-02, TAILOR-03, TAILOR-05, LIBRARY-03
