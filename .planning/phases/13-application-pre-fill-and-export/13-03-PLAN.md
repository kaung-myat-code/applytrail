Wire the CreateApplicationModal (built in plan 13-02) into its two trigger points — automatically after saving a tailored resume, and manually from any resume library card — and add Export PDF / Export JSON actions to each resume library card, calling the backend routes built in plan 13-01.

Purpose: This closes the loop from "tailored resume generated" to "application created and tracked" (PREFILL-01/02/03), and gives users a way to get any resume version out of the app as a portable file (EXPORT-01/02).

</objective>
